About ABC

We are open sourcing the ABC source code as a standalone application that runs locally on your computer. This powerful tool enables you to:

  • Create new projects with ease
  • Generate links between project components
  • Generate comprehensive documentation from your projects
  • Manage trace links and artifacts collaboratively

Getting Started

We provide complete project files that allow you to import the ABC codebase and all its requirements as a single project into the desktop app.

Quick Start Guide

1. Download the latest version of the ABC project
2. Upload it to your ABC desktop app
3. Make modifications to artifacts or trace links as needed
4. Download your modified project state
5. Submit your changes for review

Instructions for downloading and running ABC on your computer are available in the GitHub repository below.

Collaborative Change Management

This project is designed for collaborative feedback. Once you've made modifications to the ABC project in your local desktop app, you can download the state of your project and submit your changes for review to be approved into the published dataset.
